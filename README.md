### Description

- **EmotionDetection/** – Main Python package containing the emotion detection logic.  
- **static/** – Contains frontend JavaScript files used by the web interface.  
- **templates/** – HTML templates for rendering the web pages.  
- **server.py** – Backend server file that runs the web application.  
- **test_emotion_detection.py** – Unit tests for validating the emotion detection module.  
- **README.md** – Documentation for the project.  
- **LICENSE** – Project license file.  
- **.gitignore** – Specifies files ignored by Git.  
- **pylint** – Configuration or output related to pylint code analysis.  
- **__pycache__/** – Python cache directory for compiled files.
